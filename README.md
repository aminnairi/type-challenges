# type-challenges

## Requirements

- Git
- Docker
- Docker Compose
- GNU/Make

## Clone

```bash
git clone https://github.com/aminnairi/type-challenges.git ~/git/github.com/aminnairi/type-challenges
cd ~/git/github.com/aminnairi/type-challenges
```

## Installation

```bash
make install
```

## Clean Node artifacts

```bash
make clean
```
